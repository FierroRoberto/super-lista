🔑 Obtener Access Token

El token de Google autoriza a la app para leer y escribir tu archivo de estado en Drive. Dura ~1 hora; renuévalo cuando el punto de sync se ponga rojo.

1
Toca "Abrir OAuth Playground" — se abre en el navegador
2
En la lista izquierda busca Drive API v3 → marca drive.file → clic "Authorize APIs"
3
Inicia sesión con tu cuenta Google (la misma donde está el archivo)
4
Clic en "Exchange authorization code for tokens"
5
Copia el Access token (empieza con ya29.) y pégalo abajo
